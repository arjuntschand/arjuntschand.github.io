# arjuntschand.github.io

My personal site. Live at [arjuntschand.github.io](https://arjuntschand.github.io/).

Built with Vite, React, TypeScript, and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

Build the production bundle with `npm run build`.

## Deploy

Pushes to `main` build and publish via GitHub Actions.
